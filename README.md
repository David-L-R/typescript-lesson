## Main types

The main types will be the same as our data types:

- string
- boolean
- number
- null
- undefined

Array we define by using a type and adding a `[]` to it
- `strings[]`
- `boolean[]`

With object we can (and should) define each property

```typescript
type Animal = {
  name: string,
  legs: number,
  friendly: boolean,
```
